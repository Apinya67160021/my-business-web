# Apinya shop Website

เว็บไซต์ Apinya shop เป็นเว็บไซต์แนะนำร้านเสื้อผ้าผู้หญิง  
พัฒนาเพื่อฝึกการสร้างเว็บไซต์ด้วย **HTML และ CSS**  
โดยออกแบบให้มีหลายหน้า เชื่อมโยงถึงกัน และมีโครงสร้างที่เป็นระเบียบ

---

## วัตถุประสงค์ของโปรเจกต์

- เพื่อฝึกเขียนโครงสร้างเว็บไซต์ด้วย HTML5
- เพื่อฝึกการจัดหน้าเว็บและตกแต่งด้วย CSS
- เพื่อฝึกการแยกไฟล์ HTML และ CSS ออกจากกัน
- เพื่อฝึกการเชื่อมโยงหลายหน้า (Multi-page website)

---

## โครงสร้างไฟล์

project-folder/
│
├─ index.html หน้าแรกของเว็บไซต์
├─ about.html หน้าเกี่ยวกับร้านและทีมงาน
├─ services.html หน้าบริการและแพ็กเกจ
├─ contact.html หน้าติดต่อและฟอร์มติดต่อ
├─ style.css ไฟล์ CSS สำหรับตกแต่งเว็บไซต์
├─ README.md คำอธิบายโปรเจกต์
└─ images/ โฟลเดอร์รูปภาพ

## รายละเอียดแต่ละหน้า

### index.html

- แสดงโลโก้และชื่อร้าน
- เมนูนำทาง (Navigation)
- ส่วนแนะนำร้าน (Hero section)
- แสดงบริการและสินค้าเด่น
- Footer แสดงลิขสิทธิ์

### about.html

- เรื่องราวของร้าน Apinya shop
- แนะนำทีมงานพร้อมรูปภาพ
- พันธกิจ (Mission) และวิสัยทัศน์ (Vision)

### services.html

- แสดงรายการบริการของร้าน
- ตารางเปรียบเทียบแพ็กเกจการแมตช์ลุค
- ใช้ fieldset และ table ในการจัดข้อมูล

### contact.html

- ฟอร์มติดต่อ (Contact Form)
- ข้อมูลที่อยู่และช่องทางติดต่อ
- แผนที่ Google Map

## การใช้ CSS

- แยกไฟล์ CSS ออกจาก HTML (`style.css`)
- ใช้ Flexbox จัดวางเลย์เอาต์
- ใช้ class และ element selector
- ใช้ box model (margin, padding, border)
- ใช้ hover effect เพิ่มความน่าสนใจ
